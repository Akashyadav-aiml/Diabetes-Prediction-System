About This System

üéØ Overview

The Diabetes Prediction System is a machine learning application that uses advanced algorithms to predict the likelihood of diabetes based on medical data. It's designed to assist healthcare professionals in identifying at-risk patients.

ü§ñ Machine Learning Model

Algorithm: Support Vector Machine (SVM)

Kernel: Linear

Training Data: PIMA Diabetes Dataset (768 records)

Training Set: 614 samples (80%)

Test Set: 154 samples (20%)

Expected Accuracy: ~77%

üìä Input Features

1. Pregnancies - Number of times pregnant
2. Glucose - Plasma glucose concentration (mg/dL)
3. Blood Pressure - Diastolic blood pressure (mm Hg)
4. Skin Thickness - Triceps skin fold thickness (mm)
5. Insulin - 2-Hour serum insulin (mu U/ml)
6. BMI - Body mass index (kg/m¬≤)
7. Diabetes Pedigree Function - Genetic predisposition
8. Age - Age in years

üíæ Data Storage

All predictions are securely stored in a SQLite3 database. This allows for complete prediction history tracking, statistical analysis, and feedback collection for model improvement.
